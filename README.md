
# JavaScript Welcome App

A simple web application that greets users based on their input and validates the entered name.

## :sparkles: Features

- User name input
- Welcome message
- Input validation
- Short name detection
- Long name detection
- Personalized welcome message
- Responsive design
- Smooth hover and transition effects

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## :books: What I Learned

- DOM Manipulation
- Input value
- Event Listener
- Functions
- if / else
- String methods (trim())
- Form validation

## :dart: Project Goal

Practice working with user input and validation using JavaScript.

## :rocket: Future Improvements

- Press Enter to submit
- Dark/Light mode
- Display greeting with animation
- Save user name using Local Storage
